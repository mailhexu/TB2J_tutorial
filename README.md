## Example files for TB2J tutorial. 

The contains the material for running the TB2J tutorial. 

* bccFe_Siesta: example of running TB2J-Siesta for collinear spin.

- SrMnO3_Wannier: example of running TB2J-Wannier90 for collinear spin calculation.
- CrI3_Siesta: example of running TB2J-Siesta for non-collinear spin.
- CrI3_Wannier: example of running TB2J-Wannier90 for non-collinear spin.

In each directory, there is a README.md file for the details of how to run the examples. 

It is recommended to read the online documentation or follow the video tutorials.  The video tutorials were created with these files.
